Welcome to SHEAP's documentation!
=================================

**SHEAP** (Spectral Handling and Estimation of AGN Parameters) is a Python package for AGN spectral modeling, emission line decomposition, and uncertainty-aware parameter inference using modern numerical tools.

.. note::
   This documentation is a work in progress. Some modules and examples are still under development.
